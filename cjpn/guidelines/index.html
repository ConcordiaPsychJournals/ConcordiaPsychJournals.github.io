To submit an article to CJPN, please abide by the following instructions.

Manuscript Submission Format

The manuscript must be submitted as a .docx document.
The length of the text should be no longer than 4000 words. This maximum length does not include the cover page or reference list.

The format of your manuscript must conform to the following specifications:

  • 8 1/2'' x 11" page
  • Margins must be a minimum of 1.9cm on all sides
  • Indicate the page number on the bottom right corner of each page
  • Continuously double spaced text
  • Left aligned
  • 12 point font size
  • Times New Roman font type
  • In text-citations must be numbered in square brackets (e.g., as in previous work[1])
  • Using square brackets, list references by the order of appearance in the text
  • Citations included in the reference list should use APA Style formatting

Attachments Submission Format

Attachments might include (but are not limited to): figures, tables, and graphical abstracts.
Each file should be appropriately labeled.

Tables and Figures
Format Tables and figures in APA Style
Only include tables and figures that relate directly to the results that are being reported in the manuscript.
DO NOT INCLUDE supplemental graphs and figures.
Label figures as “LASTNAME_Fig_#” and tables as “LASTNAME_Table_#”

.tiff, .png, .jpg, and .pdf formats will be accepted.

Graphical Abstracts
Graphical abstracts are not mandatory, but they are encouraged!
If you wish to provide a graphical abstract, please submit it along with your manuscript.

Label graphical abstracts as “LASTNAME_VisualAbstract”
.png, .jpeg or .tiff formats will be accepted.

Forms

Publishing Agreement
The publishing agreement is a contract between the journal and the author. Please read it thoroughly and sign the document.

Principal Investigator Approval
The Principal Investigator (PI) approval document is an agreement between the journal, the author, and the investigator whose data is included within the manuscript. Please read it thoroughly, sign the document, and have the PI do the same.

These forms may be printed, signed, and scanned. Conversely, the signatories may choose to provide an electronic signature. Both methods of signing the document are acceptable.

Submission Checklist:

1. Manuscript
Manuscripts should include: Introduction, Methods, Results, Discussion
Only .docx format will be accepted.
Note: Only one manuscript submission per person will be considered per year.

2. Attachments
Create a zip file with all of the attachments that accompany your submission.

3. Publishing Agreement
Please read, print, sign and attach this document to your submission.
Only .pdf format will be accepted.
E-signatures will be accepted.

4. Principal Investigator Approval
Please read, print, sign and have your supervisor sign this document.
Only .pdf format will be accepted.
E-signatures will be accepted.

If you have any questions about the aforementioned documents, please email us at : ConcordiaJPN@gmail.com.
